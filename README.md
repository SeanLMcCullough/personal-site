# personal-site
My personal website

## Local Development

To preview and develop the site locally:

1. Install [Ruby 3.2.2](https://www.ruby-lang.org/en/downloads/) (see `.ruby-version`).
2. Install [Bundler](https://bundler.io/) if not already installed:
   
   gem install bundler

3. Install dependencies:
   
   bundle install

4. Serve the site locally with live reload:
   
   bundle exec jekyll serve --livereload

5. Open [http://localhost:4000](http://localhost:4000) in your browser to preview.

All theme and content changes will be reflected live. For more details, see THEME_GUIDE.md and CLAUDE.md.
